# Contributing

Thanks for helping improve the PyE Community bot.

## Setup

```bash
pnpm install
cp .env.example .env
```

## Local validation

```bash
pnpm lint
pnpm typecheck
pnpm build
```

## Development

```bash
pnpm dev
```

## PR checklist

- [ ] Lint, typecheck, and build pass locally.
- [ ] New behavior includes docs updates in README when needed.
- [ ] Changes are scoped and include clear rationale in PR description.

## Commit style

Prefer Conventional Commits (`feat:`, `fix:`, `docs:`, `chore:`).
